## Getting Started

Set up .ENV
```bash
TAILWIND_MODE=watch
SESSION_SECRET=""
USERNAME = ""
PASSWORD = ""
DB_URI = ''
```

First, run the development server:

```bash
npm run dev
```

and run for Connect with API

```bash
nodemon nodemon ./src/index.js
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
